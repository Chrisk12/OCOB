Player Shortcut Keys

The following shortcut keys are available in Mediasite Player: General, On-Demand Playback

General shortcut keys

The general shortcut keys are available for live and on-demand presentations. Some keyboard shortcuts are not available in all Silverlight Player layouts and depend on the features exposed in the Player. These shortcuts are indicated with an asterisk (*).

Shortcut Keys

Command

H

Open the Player Help Menu.

A

Ask a question.*

E

Share a presentation using e-mail.*

L

View the presentation's links.*

P

View polls.*

I

View presentation information.*

Ctrl+Up Arrow
(Ctrl+ ↑)

Volume up.

Ctrl+Down Arrow
(Ctrl+↓)

Volume down.

M

Mute the presentation's audio.

On-demand playback shortcut keys

The playback shortcut keys will work only when you are viewing on-demand presentations.

Shortcut Keys

Command

Space bar

Play or pause the presentation.

Left Arrow (←)

Skip back 3 seconds in the presentation.

Right Arrow (→)

Skip forward 3 seconds in the presentation.

Ctrl+Left Arrow
(Ctrl+ ←)

Go to the previous slide.

Ctrl+Right Arrow
(Ctrl + →)

Go to the next slide.
