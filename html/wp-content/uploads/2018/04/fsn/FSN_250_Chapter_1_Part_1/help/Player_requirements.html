Player Requirements

Sonic Foundry recommends the following system requirements for the best presentation viewing experience with Mediasite Player on Windows, Mac OS, Linux.

Before you begin, here are some things to consider:

  • Silverlight 3.0 is required to view presentations in the Silverlight Player. If you do not have Silverlight 3.0, you will be prompted to install it when you view a presentation. If you are not prompted, you can download it from here.
  • The Silverlight Players do not support playback speed control.
  • All Silverlight Players support closed-captioning. However, the small Silverlight Players only supports closed-captioning when in full screen.
  • If you are unable to play the presentation, contact your Mediasite administrator to make sure that HTTP streaming is enabled on the media server on port 80.
  • Remember to configure pop-up blockers to allow site pop-ups. Otherwise, presentations will not appear. Also, if you are not using Internet Explorer as your browser on Windows, there are additional considerations.

Windows requirements
  • Microsoft® Windows XP, Windows Server 2003, Windows Server 2008 (requires the Desktop Experience), Windows Vista®, or Windows 7
  • Microsoft Internet Explorer® 6.0 SP1 or later, Firefox® 2.0 or later, or Google Chrome 1.0 or later
  • Microsoft Silverlight® 3.0 or later
  • Broadband Internet connection (256 Kbps & above)

Mac requirements
  • Apple® Mac® OS X 10.4.8 or later
  • Apple Safari™ 2.0.4 or later or Firefox 2.0 or later
  • Microsoft Silverlight® 3.0 or later
  • Broadband Internet connection (256 Kbps & above)

Top

Linux® requirements
  • Moonlight™ 1.0
  • Microsoft Media Pack for Moonlight
  • Moonlight 1.0 supports the operating systems and platforms listed here.
  • Broadband Internet connection (256 Kbps & above)

Notes:

  • Moonlight is an open source implementation of Microsoft Silverlight for Unix systems.
  • The first time that Moonlight loads a page that contains media, it will ask you whether you want to install the Microsoft Media Pack.

For more information about Moonlight go here.

Additional considerations

To view Mediasite presentations using Firefox over Windows Vista or Windows XP:

  • Use the ActiveX Plug-in you are prompted to download when you first attempt to view a presentation.


    • OR

  • Copy the Windows XP plug-in dlls to your machine following these instructions.

If you are having problems and your computer meets the system requirements, visit Sonic Foundry's Troubleshooting Mediasite Playback web resource for tips on troubleshooting your computer and more information on Player features.
